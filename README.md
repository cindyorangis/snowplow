# SnowPro Services
Snow Plow Business - Full Azure Project Plan

Cloud Infrastructure Engineer Portfolio Project | AZ-104 Exam Prep

## Project Overview
SnowPro Services is a fictional snow plow business used as a hands-on Azure learning environment. The goal is to simulate a real production web application — complete with customer-facing pages, an admin portal, a quote/booking backend, and cloud monitoring — while building every Azure skill tested on the AZ-104 exam.

This project is structured into 5 progressive phases. Each phase delivers a working milestone AND maps directly to AZ-104 exam domains, so studying and building happen simultaneously.

## Your Role
Job Title: Cloud Infrastructure Engineer (Independent Project)

You will wear three hats throughout this project:
- Cloud Engineer — Design, deploy, and manage all Azure infrastructure
- Frontend Developer — Build the React website and admin dashboard
- Simulated Business Owner — Act as both customer and admin to test real workflows

## Phase 1 - Azure Foundations & Hosting
DURATION: 2-3 WEEKS | GOAL: GET THE REACT SITE LIVE ON AZURE

### What You'll Build
- React frontend for SnowPro Services (homepage, services, pricing, contact)
- Deploy to Azure Static Web Apps — free tier, auto SSL, global CDN
- Organize everything inside a Resource Group (e.g. rg-snowpro-prod)
- Set up Azure Blob Storage for storing uploaded photos (job site images, before/after)

### AZ-104 Skills Practiced
- Create and manage Resource Groups and subscriptions
- Configure Azure Storage accounts, blob containers, and access tiers
- Deploy Azure Static Web Apps via Azure Portal and CLI
- Understand Azure regions and availability concepts

A live React website at a temporary Azure URL showing: Home, Services, Pricing, and a Contact form placeholder.
